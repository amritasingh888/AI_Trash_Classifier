#  AI Trash Classifier

An AI-powered waste classification system that uses Deep Learning and Computer Vision to identify and categorize trash from images. The project helps improve waste segregation, recycling efficiency, and environmental sustainability by automatically classifying waste into different categories.

##  Features

- Upload and classify waste images
- AI-based image recognition
- Multiple waste categories detection
- Fast and accurate predictions
- User-friendly interface
- Supports sustainable waste management

##  Waste Categories

The model can classify waste into categories such as:

- Plastic
- Paper
- Cardboard
- Glass
- Metal
- Organic Waste

##  Installation

### Clone the Repository

```bash
git clone https://github.com/amritasingh888/AI-Trash-Classifier.git
cd AI-Trash-Classifier
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

**Linux/Mac**

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

## Run the Application

### For Streamlit

```bash
streamlit run app.py
```

### For Flask

```bash
python app.py
```

## Model Training

To train the model:

```bash
python train.py
```

Make sure the dataset is available in the `dataset/` directory.

## Dataset

You can use publicly available waste classification datasets such as:

* TrashNet Dataset
